# AI Chat - 智能聊天H5应用

这是一个现代化的移动端H5应用，用于与AI虚拟角色进行智能对话。应用支持多语言（中文、英文、阿拉伯语），具有完整的用户认证、角色管理、聊天功能和设置系统。

## 🌟 主要功能

### 用户认证
- 📱 用户注册和登录
- 🔐 基础的表单验证
- 💾 本地存储用户信息

### 角色管理
- 🤖 浏览公开AI角色
- ➕ 创建自定义AI角色
- 🎨 头像和人设自定义
- 🏷️ 角色标签系统

### 聊天功能
- 💬 实时1v1对话
- ✨ AI回复建议功能
- 📱 类似Telegram的聊天界面
- 🗑️ 消息删除功能

### 设置系统
- 🌍 多语言支持（中/英/阿拉伯语）
- 🆔 用户ID复制功能
- 📞 联系销售功能
- 🚪 退出登录

## 🚀 快速开始

1. **克隆或下载项目**
   ```bash
   git clone <repository-url>
   cd ai-chat-app
   ```

2. **启动本地服务器**
   ```bash
   # 使用Python启动简单HTTP服务器
   python -m http.server 8000
   
   # 或使用Node.js
   npx http-server
   
   # 或使用PHP
   php -S localhost:8000
   ```

3. **打开浏览器**
   访问 `http://localhost:8000`

## 🔧 功能说明

### 1. 用户认证
- 支持邮箱和手机号登录
- 密码强度验证
- 自动保存登录状态

### 2. 角色系统
- **公开角色**: 5个预设AI角色，各具特色
- **我的角色**: 用户自定义创建的角色
- **角色属性**: 名称、性别、年龄、头像、描述、标签

### 3. 聊天功能
- **实时对话**: 发送消息并获得AI回复
- **消息管理**: 长按删除消息
- **AI建议**: 点击✨按钮获取回复建议
- **会话管理**: 查看所有聊天记录

### 4. 设置功能
- **语言切换**: 支持中英阿三种语言
- **用户ID**: 显示并支持复制
- **联系销售**: 提交联系信息
- **退出登录**: 清除本地数据

## 🌍 多语言支持
